# Number Analyzer

A Java console application built in Java that analyzes numbers and identifies various mathematical properties.

## Current Features

- Prime Number Check
- Armstrong Number Check
- Palindrome Number Check

## Planned Features
- Perfect Number Check
- Strong Number Check
- Automorphic Number Check
- Harshad Number Check
- Neon Number Check
- Duck Number Check
- Spy Number Check
- Factor Finder
- GCD Calculator
- LCM Calculator
- Factorial Calculator
- Fibonacci Generator
- Digit Counter
- Sum of Digits

## Technologies Used

- Java
- IntelliJ IDEA
- Git
- GitHub

## Project Goal

To strengthen Java fundamentals, problem-solving skills, and software development practices by gradually building a complete number analysis toolkit.
